Соглашение Microsoft Cloud Agreement (MCA)

является транзакционным лицензионным соглашением для коммерческих и государственных организаций, стремящихся обеспечить полностью внешнее управление своими облачными сервисами через программу Cloud Solution Provider (CSP).

  1. Программа CSP в комбинации с услугами, производящими добавленную стоимость, которые предлагаются системным интегратором, партнером по размещению, торговым посредником, изначально работающим в облаке, предоставляет простой способ лицензирования облачных сервисов, необходимых вашим клиентам. Локальное программное обеспечение и Software Assurance недоступны через партнеров CSP.
