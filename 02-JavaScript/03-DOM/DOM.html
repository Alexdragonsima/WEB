DOM - Document Object Model

DOM Представляет любую HTML - страницу как сложный обьект, который агрегирует в себя другие обьекты - элементы страницы. DOM обьединяет все элементы в дерево обьектов ,корневым обьектом которого является обьект 'document'.Этот обьект всегда неявно создается в любом JavaScript - коде.
